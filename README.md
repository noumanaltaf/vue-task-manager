# vue-task-manager
Simple Task manager practice app created using <b>Vue 3 </b> and <b>Vuex 4 </b>.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
